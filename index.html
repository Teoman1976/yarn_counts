iplik numaralandırma sistemleri

  • ağırlık

    • tex

      1000 metre ipliğin gram cinsinden ağırlığı

    • dtex

      10'000 metre ipliğin gram cinsinden ağırlığı. (1 / 10) tex

    • ktex

      1000 metre ipliğin kilogram olarak ağırlığı. 1000 * tex

    • denye

      9000 metre ipliğin gram olarak ağırlığı

  • uzunluk

    • Nm

      1 gram ipliğin metre olarak uzunluğu

    • Ne

      1 lb ipliğin hank cinsinden uzunluğu

      1 hank = 840 yard = 768 metre.

  • Conversions

    • Nm = 1 / ktex

    • Nm = 1000 / tex

    • Nm = Ne x 1.69

    • Ne = Nm x 0.59

    • tex = 1000 / Nm
